Temperature
===============================

Blaney-Criddle (1950)
---------------------

.. math::

    PE=kp(0.46 * T_a + 8.13)

Hamon (1961)
------------

.. math::

    PE = (\frac{DL}{12})^2 exp(\frac{T_a}{16})

Romanenko (1961)
----------------

.. math::

    PE=4.5(1 + (\frac{T_a}{25})^2 (1  \frac{e_a}{e_s})

Linacre (1977)
--------------

.. math::

    PE = \frac{\frac{500 T_m}{(100-A)}+15 (T_a-T_d)}{80-T_a}

.. math::

    T_m = T_a + 0.006 * elevation
