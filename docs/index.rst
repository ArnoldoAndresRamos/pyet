pyet
============
*pyet* is an open source Python package for estimating reference and potential
evaporation (PET) using time series data. This enables the user to
estimate evaporation and calibrate the models with just a few lines of 
python code.  

In the current version, pyet supports eighteen PET methods to estimate ET.
